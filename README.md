# Family Budget Dashboard

A premium, responsive dashboard for managing family finances.

## Features
- **Real-time Overview**: Track total balance, income, and expenses.
- **Visual Analytics**: Interactive doughnut chart visualizing budget distribution.
- **Transaction Management**: Add new income or expense items instantly.
- **Responsive Design**: Works on desktop and mobile.

## How to Run
Simply open `index.html` in your web browser.

## Technologies
- HTML5
- CSS3 (Variables, Flexbox, Grid, Glassmorphism)
- JavaScript (Vanilla)
- Chart.js (via CDN)
